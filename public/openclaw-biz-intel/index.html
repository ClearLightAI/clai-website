02

Consolidation Layer

Where raw signals become intelligence
📊

Daily Digest

Runs at midnight UTC (8 AM Bali). Reads all new signals from the past 24 hours (~155K input tokens), produces a structured digest covering new threads, document changes, decisions, and action items. Updates living context docs.

consolidate-daily.py 0 0 * * * ~9 min runtime
📖

Weekly Narrative

Monday heartbeat appends to narrative.md — the company story in prose. Captures momentum, shifts, and patterns not visible in daily granularity.

heartbeat Mondays
◆
Key principle: Scripts handle all I/O-heavy ingestion (zero LLM cost). The agent focuses only on reasoning — reading pre-fetched content and producing intelligence. Total daily LLM spend: ~$0.02.

03

Knowledge Store

Searchable institutional memory
🔍

QMD Index

Hybrid search engine (keyword + vector + LLM re-ranker). Indexes all workspace/memory/**/*.md files automatically. 1,100+ documents indexed. Re-indexes nightly at 23:00.

hybrid search 1,100+ docs
🦎

ByteRover Context Engine

Auto-retrieves relevant knowledge before each agent turn and curates new context after. Maintains a context tree at .brv/context-tree/. Connected to OpenRouter for curation LLM.

v1.1.6 auto-curate
🧠

Living Context Documents

Five maintained files that form the agent's "situational picture." Updated by the daily consolidation cycle and queryable at any time.
